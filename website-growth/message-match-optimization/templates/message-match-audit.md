## 4. Optimization Actions

- [ ] **Mirror the Headline:** Update H1 to use at least 2-3 identical keywords
      from the source.
- [ ] **Synchronize Visuals:** Ensure the hero image matches the ad's visual
      theme or subject.
- [ ] **Validate the Offer:** Make sure any discount, price, or specific "lead
      magnet" is visible without scrolling.
- [ ] **Align the CTA:** Use the same verb in the landing page button as used in
      the source CTA.
- [ ] **Remove Distractions:** Hide navigation or footer links that lead away
      from the campaign goal.
