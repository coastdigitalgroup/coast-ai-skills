# Visual Weight Matrix

Use this template to audit your UI and ensure that elements are correctly
prioritized.

## Step 1: Priority Assignment

List your screen elements and assign them a Priority Score (1-4).

| Element Name         | Priority Score (1-4) | Goal of Element                 |
| :------------------- | :------------------- | :------------------------------ |
| Main Headline        | 1                    | Communicate value immediately   |
| Primary CTA          | 1                    | Drive the main conversion       |
| Social Proof (Logos) | 2                    | Build trust                     |
| Secondary CTA        | 2                    | Offer a lower-friction path     |
| Feature Lists        | 2                    | Explain "How it works"          |
| Footer Links         | 4                    | SEO and low-priority navigation |

## Step 2: Visual Weight Audit

For your **Priority 1** elements, check which levers you are using. You should
use at least **three** to ensure dominance.

- [ ] **Size:** Is it significantly larger than surrounding elements?
- [ ] **Color:** Does it use a high-energy or brand-specific color?
- [ ] **Contrast:** Is there a high difference between the element and its
      background?
- [ ] **Whitespace:** Is there enough "negative space" to prevent crowding?
- [ ] **Position:** Is it in a "high-value" area (Top-Center, Center, or start
      of scan path)?

## Step 3: Conflict Resolution

If two elements have the same visual weight but different priorities, use this
checklist to resolve the conflict:

1. **Reduce Priority 2:** Lighten the font weight, change color to a neutral
   gray, or reduce the font size.
2. **Increase Priority 1:** Add more whitespace around it, or give it a more
   vibrant background color.
3. **Move Priority 2:** Relocate the lower priority item further down the page
   or away from the main scan path.

## Step 4: The Squint Test Verification

- [ ] **Success:** The Priority 1 element is the first thing I see when I
      squint.
- [ ] **Success:** My eye follows a logical path from 1 to 2 to 3.
- [ ] **Success:** No "visual noise" or irrelevant elements are distracting from
      the goal.
